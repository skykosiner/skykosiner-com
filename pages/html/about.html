Progarming Joureny

I started to code at the age of 11 with python. Then at 12 went into the world of frontend web development, but after a while I got bored so I started to learn backend.


After awhile I found JavaScript annoying and a pain, so I played around with typescript and made a few projects with Typescript.

But once again I found Typescript still shared issues with JavaScript so I looked into rust, but it was too hard for me. So I decided to learn golang, and right now golang is my favorite programming languages, no matter if it's for cli tools, tui tools, api's or more.

After a while I decided to switch to vim and went linux full time on my main machine. These days I do most things in Doom Emacs writing loads of bash scripts and DWM as my window manger (Arch BTW). Overall Learning Linux was a very good decision I'm glad I made.
